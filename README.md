# Lenux

  
왜 리눅스?
-------------------------------------
1. 산업에서 대부분 사용되고 있기 때문에

2. 오픈소스의 무료화

3. 서버의 안정성

pwd 현재 위치

ls -al (--help, man) 현재 위치에 존재하는 파일 또는 디렉토리 리스트 출력

디렉토리생성
mkdir (디렉토리 이름)

파일생성
touch (파일 이름)

파일삭제
rm (filename)

디렉토리 삭제
rm -r (이름) 

파일 내용 변경수정
vi (파일네임 , i ,Shift+ZZ)

파일이름 수정
mv (기존의 파일이름) (변경할 파일이름)

파일 위치 변경
mv (파일 이름) (변경할 위치)

파일 검색
find (경로) -name (파일 이름) or (아스타리크.확장자)
